Este encontro trabalha **Botoes, cards, alertas e componentes reutilizaveis** na camada visual, conectando CSS moderno com layout real, componentes e consistencia de interface.

Componentizacao de elementos frequentes orienta a tomada de decisao tecnica e melhora a qualidade do projeto.

