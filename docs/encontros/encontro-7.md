# Encontro 7 - Imagens, tabelas e organização de conteúdo

**Unidade:** Unidade 1  
**Carga prevista:** 1,5h  
**Entregável previsto:** Página com mídia e tabela

## Explicação do conteúdo
Neste encontro você integra imagens e tabelas ao HTML com boas práticas de estrutura e interpretação de dados.

![Imagem de apoio ao tema](https://upload.wikimedia.org/wikipedia/commons/6/61/HTML5_logo_and_wordmark.svg)

## Conceitos essenciais
- Inserção de imagens com texto alternativo.
- Estrutura de tabelas (`th`, `td`, `tr`).
- Leitura de dados com clareza visual.

## Exemplo prático
Use o exemplo como ponto de partida e altere partes pequenas para entender cada efeito no navegador.

```html
<img src="laboratorio.jpg" alt="Estudantes no laboratório" width="320" />
<table>
  <tr><th>Ferramenta</th><th>Função</th></tr>
  <tr><td>VS Code</td><td>Edição de código</td></tr>
  <tr><td>DevTools</td><td>Depuração</td></tr>
</table>
```
