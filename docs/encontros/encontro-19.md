# Encontro 19 - Lista Orientada de Exercícios de CSS

**Unidade:** Unidade 1  
**Entrega:** Lista 2 (CSS)

## Visão Geral
Neste encontro, você consolida os conteúdos de CSS estudados do Encontro 12 ao 18 em uma lista orientada de exercícios.
O foco é verificar domínio técnico de seletores, propriedades visuais, box model, layout e responsividade.

Se nos encontros anteriores você aprendeu cada tópico em etapas, agora você integra tudo em atividades progressivas antes da Avaliação 02.

## Conceitos Essenciais
- Organização de arquivo `styles.css`.
- Uso consciente de seletores e hierarquia de regras.
- Aplicação de tipografia, cor e espaçamento.
- Construção de layout com Flexbox ou Grid.
- Adaptação responsiva com media query.

## 1) Objetivo da Lista 2
A Lista 2 avalia sua capacidade de estilizar páginas HTML já estruturadas, mantendo clareza visual e organização técnica.

### Competências avaliadas
- conectar HTML e CSS corretamente;
- usar seletores simples e combinados;
- aplicar propriedades visuais coerentes;
- organizar componentes com box model;
- montar layout com Flexbox ou Grid;
- aplicar responsividade básica.

## 2) Escopo da Lista

### Bloco A - Fundamentos de CSS
- arquivo externo `styles.css`;

### Bloco B - Aparência visual

- meta viewport;
- imagens fluidas;
- ao menos uma media query.

## 5) Lista orientada de exercícios

### Exercício 1 - Estilização básica de página textual
Use uma página HTML simples e aplique:
- tipografia para `body`, `h1` e `h2`;
- pelo menos 3 cores;
- fundo para `header` e `footer`;
- classe de destaque para um parágrafo;
- espaçamentos com `margin` e `padding`.

### Exercício 2 - Cartões com box model
Monte uma página com:
- contêiner centralizado;
- pelo menos 3 cartões;
- borda, `padding` e `margin`;
- links ou botões com `display: inline-block`.

### Exercício 3 - Layout com Flexbox ou Grid
Crie uma página com:
- `header`, `nav`, `main` e `footer`;
- navegação organizada com Flexbox;
- área de conteúdo em Flexbox ou Grid;
- espaçamento com `gap`.

### Exercício 4 - Ajuste responsivo
Escolha um dos exercícios anteriores e torne-o responsivo:
- inclua meta viewport;
- use imagem fluida, se houver imagem;
- adicione ao menos uma media query;
- reorganize o layout para telas menores e maiores.

## 6) Validação rápida antes de considerar concluído
- Todos os `index.html` estão ligados ao `styles.css`.
- O CSS tem indentação e organização mínimas.
- Há contraste razoável entre texto e fundo.
- O layout não quebra imediatamente em tela estreita.
- Existe ao menos uma media query quando exigida.

## 7) Erros comuns de iniciantes
- misturar HTML e CSS inline sem necessidade;
- aplicar muitas cores sem hierarquia;
- deixar blocos colados por falta de espaçamento;
- usar Flexbox ou Grid sem observar o contêiner principal;
- esquecer de testar a página redimensionando a janela.
